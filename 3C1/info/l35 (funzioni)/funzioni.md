# Funzioni

Le funzioni sono dei "sotto programmi" che possono essere richiamate, si possono passare dei valori, e possono ritornare dei valori.



### Tipi

* void -- Non ritorna alcun valore
* int, float, bool, char, string, … -- Ritorna il valore del tipo dichiarato



### Dichiarazione

{tipo di funzione} {nome funzione}({tipo di dato} {nome variabile}){

 	// codice

 	return {valore da tornare};

}

#### 

#### Richiamazione

Per richiamare una variabile si scrive il nome della variabile con le parentesi, nelle quali si inseriscono i dati da dare

in input alla funzione, se c'è da farlo.



### Esempio



int addizione(int n1, int n2){

&nbsp;   return n1 + n2;

}



Questa funzione riceve 2 numeri int in input (n1, n2) e ritorna la somma tra i due (n1 + n2).
