## **Comandi Dos**



Path - percorso della directory

* dir - elenco del contenuto della directory
* cd \[Path] - cambia il Path - Es: cd ..; cd Desktop
* md \[Path] / mkdir \[Path] - crea nuova cartella
* rd \[Path] / rmdir \[Path] - rimuove la cartella
* rename \[Input] \[Output] - rinomina
* copy \[Input] \[Output] - copia file
* \[lettera unità]: - cambia unità - Es: F:
* tree - mostra l'albero della directory
* ipcofig - mostra le info della rete
* ping \[ip] - pinga un ip
* echo \[cose] - stampa il messaggio in console
* \[comando] > sasso.txt - salva l'output del comando in sasso.txt - Es: echo sasso > sasso.txt - (con >> si aggiunge l'output al file txt senza sovrascriverlo)
