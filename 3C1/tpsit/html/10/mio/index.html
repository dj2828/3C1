Le memorie presenti in un elaboratore, in ordine decrescente di capacità:

  1. Registri della CPU
  2. Cache
  3. RAM
  4. Hard Disk

Modalità di archiviazione dei dati:

  • memorie flash (es. SSD)
  • dischi magnetici (es. HDD)
  • dischi ottici (es. DVD)
  • cloud storage

Dispositivi di memoria di massa:

SSD (Solid State Drive)
dispositivo che utilizza una memoria flash;
HD o HDD (Hard Disk Drive)
dispositivo che utilizza uno o più dischi magnetizzati;
DVD (Digital Versatile Disc)
dispositivo di tipo ottico.
